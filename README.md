React Native library for requesting App Store and Google Play reviews

## Installation

```sh
npm install react-native-request-review
```

or

```sh
yarn add react-native-request-review
```

## Usage

```js
import { requestReview } from 'react-native-request-review';

// ...

requestReview();
```
